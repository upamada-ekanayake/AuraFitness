# AuraFitness

AuraFitness is a mobile-first fitness web app for gym users.

The project has two main parts:

1. AI workspace using Python
2. React TypeScript app for the user interface

The app will help users manage:

- Weekly workout routines
- Workout sessions
- Exercise progress
- Water intake
- Calories
- Fasting
- Body weight
- Rest days
- Gym streaks
- AI-based suggestions

## Development Order

1. Project folder setup
2. Documentation setup
3. Python AI workspace setup
4. Dataset research
5. Rule-based AI engines
6. AI export layer
7. React TypeScript app setup
8. UI/UX design system
9. Core features
10. Testing and bug fixes
11. PWA and Native Android Wrapper
12. Final polish

## Project Structure

- `ai/` - Python AI logic and exported data
- `app/` - React TypeScript frontend
- `docs/` - Planning, architecture, and testing documents
- `prompts/` - ChatGPT/Codex prompt history
