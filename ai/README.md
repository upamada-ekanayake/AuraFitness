# AuraFitness AI Workspace

This folder contains the Python-based AI foundation for AuraFitness.

## Goal

Start with simple rule-based AI engines first.

Later, add lightweight machine learning only where useful.

## Planned AI Features

1. Exercise recommendation engine
2. Progressive overload engine
3. Rest day suggestion engine
4. Calorie burned estimator
5. Streak insight engine
6. Motivation message generator

## Folder Structure

- `datasets/` - Raw and cleaned datasets
- `notebooks/` - Optional Jupyter notebooks
- `scripts/` - Main runnable Python scripts
- `utils/` - Reusable AI logic
- `models/` - Trained models later
- `exports/` - JSON files for the React app
